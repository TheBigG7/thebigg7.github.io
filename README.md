## 🎯 Objetivo del Proyecto

Este repositorio contiene una Single Page Application (SPA) estática diseñada para presentar mi perfil como Tecnólogo en Desarrollo de Software. La arquitectura del sitio prioriza el rendimiento, la accesibilidad y el diseño adaptativo (Responsive Design) sin depender de frameworks pesados para una carga ultra rápida.

## 🚀 Stack Tecnológico

El proyecto está construido utilizando tecnologías web nativas para garantizar el control total sobre el DOM y el renderizado:

* **Estructura:** HTML5 Semántico.
* **Estilos:** CSS3 nativo (Variables CSS, Flexbox, CSS Grid).
* **Tipografía:** Google Fonts (`Inter`, `Fira Code`).
* **Iconografía:** Font Awesome 6 (SVG/Webfonts).
* **Despliegue:** GitHub Pages (Servidor estático).

## ⚡ Características Técnicas

* **Zero Dependencies:** No utiliza Node.js, NPM, ni frameworks de CSS pesados en el empaquetado final, reduciendo la superficie de ataque y el tiempo de carga.
* **Animaciones CSS por Hardware:** Implementación de `@keyframes` para micro-interacciones (ej. botón de WhatsApp) optimizadas para no bloquear el hilo principal de renderizado.
* **Favicon SVG Inline:** Icono de pestaña renderizado mediante *Data URI*, eliminando peticiones HTTP adicionales y garantizando nitidez en pantallas de alta densidad de píxeles.
* **Descarga de CV Nativa:** Integración del atributo `download` para servir artefactos PDF directamente desde el repositorio.
